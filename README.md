## ✨ Sobre o projeto

Este portfólio foi criado inteiramente por mim para apresentar minha trajetória como desenvolvedora front-end e QA Engineer em formação.

Todo o código foi escrito manualmente, linha a linha, aplicando os conceitos que estou aprendendo na prática:

- Estrutura semântica em **HTML5**
- Estilização com **CSS3** puro (sem frameworks como Bootstrap ou Tailwind)
- Interatividade com **JavaScript** vanilla (sem jQuery ou React)

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura e semântica das seções |
| CSS3 | Layout, animações, responsividade e paleta de cores |
| JavaScript | Cursor personalizado, partículas, scroll reveal, contador animado |
| Google Fonts | Fontes Syne + DM Sans |
| GitHub Pages | Hospedagem gratuita |

---

## 🧠 Conceitos aplicados

| Conceito | Onde foi usado |
|---|---|
| `CSS Custom Properties` | Paleta de cores centralizada em `:root` |
| `CSS Grid` | Layout das seções em 2 colunas |
| `Flexbox` | Navbar, botões, cards |
| `@keyframes` | Animações de float, blob, typewriter, bounce |
| `clamp()` | Tipografia responsiva sem media queries |
| `IntersectionObserver` | Scroll reveal performático |
| `requestAnimationFrame` | Cursor suave com interpolação linear |
| `backdrop-filter` | Efeito vidro fosco na navbar |
| `CSS Variables no JS` | Efeito ripple dinâmico nas skill cards |
